A) Install Template - top

Requirement: NodeJS, npm and React base Knowledge

Create New Folder (ex: yourApp)

Unzip the downloaded Template file to a folder (ex: yourApp)

Open Terminal in this folder and write:

npm install

After installing libraries

npm start

Navigate to http://localhost:3000

To stop server press CTRL+C (com+C Mac's)

And check dependencies
